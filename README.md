Folder Structure
============================

> Folder structure for next js projects

### A typical top-level directory layout

    .
    ├── ...
    ├── .next         
    ├── .vscode        
    ├── app     
    │   ├── api  
    │   │     ├──  preview.ts           
    │   │     ├──  exit-preview.ts         
    │   ├── page.tsx (Home Page)            
    │   ├── layout.tsx (Layout Page)            
    ├── components           
    ├── node_modules         
    ├── public            
    ├── style                    
    │   ├── global.css          
    │   ├── global.scss       
    ├── types                    
    │   ├── index.ts                                                      
    │── .eslintrc.json
    │── .gitignore
    │── next.config.js
    │── package-lock.json
    │── package.json
    │── postcss.config.js
    │── README.md
    │── tailwind.config.js
    │── tsconfig.json

> Use short lowercase names at least for the top-level files and folders except
> `LICENSE`, `README.md`
